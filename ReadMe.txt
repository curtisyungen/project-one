# Grocery List
### Users can find recipes and generate grocery lists for them

</br>



<br/>

### Web Use

Simply visit **[this heroku link](https://recipe-to-grocery-list.herokuapp.com/)** and find recipes to shop for!

<br/><br/>

### Local Use
Command-Line:
1. git clone https://github.com/curtisyungen/Grocery-List)
2. cd project-one
3. npm install
4. node server.js

Browser:
1. visit localhost:3000

<br/><br/>
